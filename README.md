# Foxentry BE

## Description

API made with [Nest](https://github.com/nestjs/nest) & [elastic](https://www.elastic.co/) for e-commerce inventory management.

Deployed at [Vercel](https://vercel.com/)

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm start

# watch mode
$ pnpm start:dev

# production mode
$ pnpm start:prod
```
